# just1jray.com

Personal landing page for Just 1 J. Ray. Built with [Astro](https://astro.build) and [Bun](https://bun.sh).

## Development

```sh
bun install
bun run dev
```

Opens at `localhost:4321`.

## Deploy

Hosted on [Cloudflare Pages](https://pages.cloudflare.com). Pushes to `main` trigger automatic deploys.
